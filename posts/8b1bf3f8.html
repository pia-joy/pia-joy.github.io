零基础安装 Hexo 完整教程

Hexo 是一款基于 Node.js 的快速、简洁且高效的静态博客框架，支持 Markdown 语法，只需简单命令即可生成精美静态页面，搭配 GitHub Pages 可免费搭建个人博客，无需服务器、零成本上线。本文以 Windows 系统 为例，带你一步步完成 Hexo 安装与本地初始化。

一、安装前准备（必备环境）

Hexo 运行依赖两个核心工具，必须先安装，否则无法正常使用：

  1. Node.js（≥14.0 版本，推荐 LTS 稳定版）
  2. Git（版本管理工具，用于后续部署）

1. 安装 Node.js

  • 打开 Node.js 官网：https://nodejs.org/
  • 下载 LTS 长期支持版（左侧绿色按钮），一路默认安装即可。
  • 验证安装：按下 Win+R 输入 cmd 打开命令提示符，执行：
  • 出现版本号即安装成功。

2. 安装 Git

  • 打开 Git 官网：https://git-scm.com/
  • 下载对应系统版本，默认安装即可。

五、常用 Hexo 命令（收藏备用）

  • hexo init 文件夹名：初始化博客
  • hexo new "文章标题"：新建文章
  • hexo generate / hexo g：生成静态页面
  • hexo server / hexo s：启动本地服务
  • hexo clean：清理缓存文件（页面异常时使用）

六、常见问题解决

  1. npm 安装慢：切换淘宝镜像
  2. 权限报错：关闭 cmd，重新以管理员身份运行。
  3. 端口被占用：指定端口启动
